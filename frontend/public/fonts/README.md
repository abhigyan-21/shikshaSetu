# OpenDyslexic Font

This directory should contain the OpenDyslexic font files for accessibility support.

## Installation

1. Download OpenDyslexic font from: https://opendyslexic.org/
2. Place the `OpenDyslexic-Regular.woff2` file in this directory

## Alternative

If the font file is not available, the application will fall back to the system's default sans-serif font.
